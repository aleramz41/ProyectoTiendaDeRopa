Proyecto Tienda De Ropa

Este programa de escritorio gestiona clientes, productos y las ventas de una tienda de ropa.
Desarrollado en Java con interfaz gráfica usando Swing y base de datos PostgreSQL.

Requerimientos

- Registrar clientes
- Editar clientes
- Eliminar clientes
- Registrar productos
- Editar productos
- Eliminar productos
- Registrar ventas seleccionando un cliente y los productos
- Calcular el total
- Ver historial de ventas.

Tecnologías

- Java 17
- Java Swing
- PostgreSQL (Neon)
- Maven

Ejecución

1. Clonar el repositorio (git clone https://github.com/aleramz41/ProyectoTiendaDeRopa.git)
2. Abrir el proyecto en el IDE de Java de tu elección como proyecto Maven
3. Ejecutar Main.java

Estructura del proyecto

El proyecto está dividido en cuatro paquetes: model que contiene las clases de datos (Cliente, Producto, Ventas), 
repository que maneja el acceso a la base de datos, service que tiene la lógica del negocio y las validaciones, y view 
que contiene las ventanas de la interfaz gráfica.
